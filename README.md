🚀 Getting Started (Local Setup)
Clone the repo
git clone https://github.com/your-username/tic-tac-toe.git
cd tic-tac-toe

Install dependencies

Client (Frontend)

cd client
npm install
npm start


Server (Backend)

cd server
npm install
npm start

Run the game

Frontend → http://localhost:3000

Backend → http://localhost:5000
